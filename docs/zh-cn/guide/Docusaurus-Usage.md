## 1. 环境
- Node.js 16.14+

```shell
node -v

npm -v
```

## 2. 初始化
```shell
npx create-docusaurus@latest light-docusaurus classic --typescript

# 或者
npm init docusaurus ./light-docusaurus classic --typescript
```

## 3. 运行
```shell
npm run start
```
